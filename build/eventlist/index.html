HI-FIVE　PROJECT

TABLE FOR TWO(以下TFT)とは、先進国の肥満と途上国の飢餓の不均衡を正すことを目指すNPOです。TFTが考えるヘルシーメニューを食堂に導入し、それに含まれる20円がアフリカの子供たちの給食一食分として寄付されます。私たちはこの活動に賛同し、TFT大学連合の獨協支部として活動しています。
